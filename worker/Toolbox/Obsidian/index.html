简介

Obsidian 是一款基于Markdown 语法编辑的笔记软件，与传统的 md 软件不同的是，它可以创建双向链接（其他的一些软件也可以，但是都收费，而obsidian完全免费）更重要是有很多优秀的插件可以支持扩展。

图床

1️⃣ Obsidian 不能像 Typora 一样自带配置进行上传，需要自己安装插件，配置后进行上传
2️⃣ 前置条件：需要有 node,npm,picgo 等环境，参照插件的 README 文档进行配置

在插件中心进行搜索并下载 Image auto upload Plugin

picgo-core 的介绍里面说的 node

引用站外地址
picgo-core
https://picgo.github.io/PicGo-Core-Doc/zh/guide/getting-started.html

picgo-core 安装好后需要配置图床的信息

直接再Obsidian 中粘贴图片就会上传到配置的目标地址了,如果是Mac 系统上传报错把这个开关打开后重启软件再试试

发布文章

有时候需要快速的将文章通过网络的形式发布给别人看，我们就可以使用 Obsius Publish 这个插件

1️⃣ 注意事项：包含敏感信息的文章不要发布到网上，尤其带有个人或公司账户信息的
2️⃣ 如果选了这个 Remove from Obsius 后就无法在进行访问了，会报 404

统计笔记

File Explorer Note Count插件是一个能统计当前文件夹下有多少笔记并展示出来的插件

Editing Toolbar
